## Самостоятельный аудит интерфейса от 22.06.2026

Проверка выполнялась через окно браузера Codex. Целевой стенд `ttm-mini` по адресу `https://172.17.16.50:8444/` открыть не удалось из-за ошибки сертификата `ERR_CERT_AUTHORITY_INVALID`: встроенный браузер не дал безопасного способа принять сертификат. Для просмотра интерфейса была поднята локальная копия проекта на `http://localhost:8088`, локальная БД синхронизирована через `prisma db push` и заполнена demo seed.

Проверенные роли:

- `dispatcher.demo` / `dispatcher`: план производства, терминал участка, граф процесса, номенклатура, справочники, смены и KPI, партии, импорт Excel, заказы, архив.
- `admin.demo` / `admin`: дополнительно проверены дашборд директора и исполнители.

Ориентировочные локальные замеры:

| Действие | Время |
| --- | --- |
| Перезагрузка приложения | 1173 мс |
| Переход на план производства | 607 мс |
| Переход на граф процесса | 583 мс |
| Переход на номенклатуру | 595 мс |
| Переход на партии | 585 мс |
| Переход на дашборд директора | 595 мс |

Важно: эти цифры не заменяют замер на сервере `ttm-mini`, потому что локальная сеть, объем данных, TLS и серверная БД отличаются.
